## Development

### Prerequisites

- Node.js ≥ 22.12.0 (use `.nvmrc` with `nvm use`)

### Install

```bash
npm ci
```

### Run

```bash
npm run dev
```

Opens `index.html` at http://localhost:8001.

### Build

```bash
npm run build
```

Concatenates `src/js/*.js` → `script.js` and compiles `src/css/styles.scss` → `styles.css`.

### Lint & format

```bash
npm run lint
npm run format
```

### Test

```bash
npm test
```
